# PyroSim Source Models

All `.psm` files are kept here so source models do not get mixed into model-ready CSV datasets.

```text
train_data/
  a_1MW.psm
  b_1MW.psm
  ...
  j_1MW.psm
test_data/
  fst_3MW.psm
  fst_6MW.psm
  fst_8MW.psm
```

`data/train_data` and `data/test_data/fire_surface_test/model_ready` contain CSV files used by the model loader. This folder contains PyroSim source models only.
